Module 1: Personal Developer Website


-----------------------------
How to Run This Website
-----------------------------

1. Open a terminal in VS Code and type in the following to go into the module_1 folder:
   cd module_1

2. Install required packages:
   pip install -r requirements.txt

3. Run the Flask application:
   python run.py

4. Open the website in your browser using:
   http://127.0.0.1:8080
