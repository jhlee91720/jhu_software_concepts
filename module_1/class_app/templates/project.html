Module 1: Portfolio Website

Description

This project is a personal developer website built using Flask. It introduces my background and provides contact info and project links. Working with Flask, a micro web framework written in Python, I was able to learn to work with HTML, and cascading style sheets (CSS) to describe how HTML elements are to be displayed. The output is a functional personal website that includes details about you, your projects, and contact information.

Tools Used

Flask, HTML, CSS, Bootstrap, GitHub, Searching Methods
